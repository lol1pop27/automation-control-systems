#  САУ РЗДРТ

**Система автоматизированного управления рентгеновскими зеркалами, детекторами и рентгеновскими трубками**  
(калибровочный стенд «РЭКС»)

##   Описание

САУ РЗДРТ — это система управления шаговыми двигателями калибровочного рентгеновского стенда «РЭКС».  
Она обеспечивает удалённое позиционирование элементов оптической системы:

- рентгеновские зеркала  
- детекторы  
- рентгеновская трубка  
- счётчик  

Управление осуществляется с ПК через COM-порт с использованием ПО **Shag4.exe**.

---

##   Архитектура системы

Система включает:

-   Блок коммутации  
-   Блоки контроллеров (×2)  
-   Шаговые двигатели (ШД)  
-   Источник питания +12 В  
-   ПК с COM-портом  

---

##   Принцип работы

1. ПК отправляет команды через **RS-232 (COM-порт)**
2. Блок коммутации передаёт сигналы контроллерам
3. Контроллеры управляют шаговыми двигателями
4. ШД перемещают элементы стенда «РЭКС»

  Диапазон перемещения: **−10° … +190°**

---

##   Требования к ПК

| Параметр | Требование |
|----------|-----------|
| CPU | Intel Core i3 2.2 GHz или выше |
| RAM | ≥ 2 GB |
| GPU | ≥ 2 GB |
| HDD | ≥ 500 MB |
| ОС | Windows |
| COM-порт | 1 шт. |
| Разрешение | 1024×768 @ 60 Hz |

---

##   Подключение

Система подключается следующим образом:

- ПК ↔ Блок коммутации (COM-порт)
- Блок коммутации ↔ Контроллеры
- Контроллеры ↔ Шаговые двигатели
- Контроллеры ↔ Источник питания 12 В

---

##   Запуск системы

1. Подключить все устройства согласно схеме  
2. Подать питание **+12 В** на контроллеры  
3. Включить ПК  
4. Запустить `Shag4.exe`  
5. Выбрать COM-порт  

  Программа должна запускаться **через 5 секунд после подачи питания**

---

##   ПО Shag4.exe

Программное обеспечение обеспечивает:

- управление шаговыми двигателями  
- настройку углов элементов системы  
- сохранение и восстановление положений  
- визуализацию стенда  
- работу в интерактивном и автономном режимах  

---

##   Интерфейс связи (RS-232)

### Параметры порта

| Параметр | Значение |
|----------|----------|
| Скорость | 19200 |
| Формат | 8N1 |
| Порт | COM1 |

---

### Формат сообщения

$ + Address + Command + DataLength + Data

| Поле | Описание |
|------|----------|
| `$` | Начало сообщения |
| Address | Адрес контроллера (0–4) |
| Command | 2 символа команды |
| DataLength | Количество байт данных |
| Data | Параметры команды |

---

### Адреса устройств

| Адрес | Устройство |
|------|------------|
| 1 | Зеркало №1 |
| 2 | Рентгеновская трубка |
| 3 | Поворотный стол |
| 4 | Счётчик |

---

##   Функции управления

-   пошаговое вращение  
-   аварийная остановка двигателей  
-   установка текущего положения как «ноль»  
-   возврат в нулевое положение  
-   блокировка управления элементами  

---

##   Условия эксплуатации

- Температура: **0 … +35 °C**  
- Влажность: **до 80% (при 20 °C)**  
- Давление: **630–800 мм рт. ст.**  
- Только стационарные помещения  

---

##   Безопасность

- Запрещено вскрытие блоков без разрешения производителя  
- Обслуживание только квалифицированным персоналом  
- Соблюдать правила работы с ПК и электрооборудованием  

---

##   Хранение и транспортировка

- Температура хранения: **−20 … +35 °C**  
- Защита от влаги, пыли и агрессивных сред  
- При длительном хранении использовать защитный чехол  

---

##   Технические характеристики

- Питание: **12 ± 1 В**
- Потребление: **до 6 А**
- Интерфейс: **RS-232**
- Управление: шаговые двигатели с редуктором  
  *(1 шаг = 1 угловая минута)*

---

##   ПО в составе

---

##   Примечания

- Система сохраняет положение осей при закрытии ПО  
- Автоматически восстанавливает состояние при запуске  
- Предназначена для калибровочного стенда «РЭКС»
