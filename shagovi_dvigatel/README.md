САУ РЗДРТ

Система автоматизированного управления рентгеновскими зеркалами, детекторами и рентгеновскими трубками (калибровочный стенд «РЭКС»)

📌 Описание

САУ РЗДРТ — это система автоматизированного управления шаговыми двигателями рентгеновского калибровочного стенда «РЭКС».
Позволяет дистанционно управлять положением оптических элементов: зеркал, детектора, рентгеновской трубки и счётчика через ПК.

Управление осуществляется через ПО Shag4.exe по интерфейсу RS-232 (COM-порт).

⚙️ Состав системы

Система включает:

Блок коммутации А1317-Л2800.80.25.100
Блоки контроллеров А1317-Л2800.80.25.300 (×2)
Жгуты коммутации и питания
Шаговые двигатели (ШД)
Источник питания +12 В
ПК с COM-портом
🧠 Принцип работы
ПК отправляет команды через COM-порт (RS-232)
Блок коммутации передаёт команды контроллерам
Контроллеры управляют шаговыми двигателями
ШД перемещают элементы оптической системы

Ограничение перемещения:

от -10° до +190°
💻 Требования к ПК
CPU: Intel Core i3 2.2 GHz или выше
RAM: ≥ 2 GB
GPU: ≥ 2 GB
HDD: ≥ 500 MB
OS: Windows
COM-порт: 1 шт.
Разрешение: 1024×768 @ 60 Hz
🔌 Подключение

Основные соединения:

Блок коммутации → блоки контроллеров (через жгуты)
Контроллеры → шаговые двигатели
Контроллеры → источник питания 12 В
Блок коммутации → ПК (COM-порт)
🚀 Запуск системы
Подключить оборудование согласно схеме
Подать питание 12 В на контроллеры
Включить ПК
Запустить Shag4.exe
Выбрать COM-порт в программе

⚠️ Важно: запуск программы выполнять через 5+ секунд после подачи питания контроллеров

🧩 Описание ПО Shag4.exe

Программа обеспечивает:

Управление шаговыми двигателями
Настройку углов элементов системы
Сохранение и восстановление положений
Визуализацию положения стенда
Работа в интерактивном и автономном режимах
📡 Протокол обмена (RS-232)
Параметры порта
Скорость: 19200
Формат: 8 bit, 1 stop, no parity
COM1 (по умолчанию)
Формат сообщения
$ + Address + Command + DataLength + Data
Поле	Описание
$	Начало сообщения
Address	Адрес контроллера (1–4, 0 = broadcast)
Command	2 символа команды
DataLength	Размер данных
Data	Параметры команды
Адреса устройств
1 — зеркало №1
2 — рентгеновская трубка
3 — поворотный стол
4 — счётчик
🎛 Функции управления
Поворот ± шагами
Установка текущего положения как нуля
Возврат в нулевое положение
Экстренная остановка двигателей
Блокировка элементов управления
⚠️ Условия эксплуатации
Температура: 0…+35°C
Влажность: до 80%
Давление: 630–800 мм рт.ст.
Только стационарные помещения
🛑 Безопасность
Запрещено вскрытие блоков без разрешения производителя
Работа допускается только обученным персоналом
Соблюдать правила работы с ПК и электрооборудованием
📦 Хранение и транспортировка
Температура хранения: −20…+35°C
Защита от пыли, кислот, щелочей и агрессивных газов
При длительном хранении — закрывать защитным чехлом
🛠 Технические характеристики
Питание: 12 ±1 В
Потребление: до 6 А
Интерфейс: RS-232
Управление: шаговые двигатели через редуктор (1 шаг = 1 угловая минута)
📁 ПО в составе
Shag4.exe — основная управляющая программа
config.ini — файл конфигурации
📌 Примечания
Система предназначена для калибровочного стенда «РЭКС»
Поддерживает сохранение последнего состояния при закрытии ПО
Автоматически восстанавливает позицию при запуске
