Система синхронизации, управления и мониторинга измерительного комплекса
📌 Описание

NetAgent — это программно-аппаратная система для централизованного управления, синхронизации и мониторинга распределённого измерительного комплекса.

Система объединяет генерацию точных временных задержек, сбор аналоговых сигналов и управление дискретными каналами через промышленное оборудование, связанное по оптоволоконной сети.

Используется для задач, где требуется:

точная временная синхронизация сигналов
удалённое управление каналами
сбор телеметрии в реальном времени
🧩 Архитектура системы

🔌 Дискретный вывод: Advantech ADAM-4068
🌐 Сетевое соединение через промышленный коммутатор

Выполнить настройку каналов и задержек
