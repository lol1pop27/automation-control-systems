Система синхронизации, управления и мониторинга измерительного комплекса
📌 Описание

SAY LYCH — программно-аппаратная система, разработанная в среде LabVIEW, предназначенная для централизованного управления, синхронизации и мониторинга распределённого измерительного комплекса.

Система обеспечивает:

высокоточную временную синхронизацию сигналов
сбор и обработку аналоговых данных
управление дискретными каналами

Взаимодействие между компонентами реализовано через промышленную сеть с использованием оптоволоконной линии связи.

## 📌 Описание

**SAY LYCH** — программно-аппаратная система, разработанная в среде **LabVIEW**. Предназначена для централизованного управления, синхронизации и мониторинга распределённого измерительного комплекса.

### Что обеспечивает система:

- ✅ Высокоточную временную синхронизацию сигналов  
- ✅ Сбор и обработку аналоговых данных  
- ✅ Управление дискретными каналами  

Взаимодействие между компонентами реализовано через **промышленную сеть** с использованием **оптоволоконной линии связи**.

---

## 🧠 Технологический стек

- 🧩 **LabVIEW (National Instruments)** — основная среда разработки  
- 🌐 **Ethernet / TCP-IP**  
- 🔗 **Оптоволоконная сеть**  
- 📡 **Промышленные протоколы**:
  - Modbus/TCP
  - SCPI* (управление генератором задержек)

| Устройство | Модель | Назначение |
|------------|--------|-------------|
| 🎛 Генератор задержек | Stanford Research Systems DG645 | Формирование таймингов |

| 🔌 Дискретный вывод | Advantech ADAM-4068 | Управление исполнительными устройствами |
| 🌐 Коммутатор | Промышленный Ethernet | Сетевая маршрутизация |
| 🔗 Линия связи | Оптоволокно | Удалённое взаимодействие |
| 💻 ПК | LabVIEW (SAY LYCH) | Центральный контроллер |

## ⚙️ Основные возможности

### ⏱ Синхронизация и тайминг
- Управление генератором DG645  
- Настройка каналов задержек (A–H)  
- Точная установка задержек (наносекундный диапазон)  
- Формирование и синхронизация импульсов  

### 📡 Сбор данных
- Опрос ADAM-4017  
- Измерение токов и аналоговых сигналов  
- Отображение данных в реальном времени  

### 🔌 Управление исполнительными устройствами
- Управление ADAM-4068  
- Включение/выключение каналов  
- Управление зонами (ближняя / дальняя)  
- Контроль FO-Ethernet линий  

### 🌐 Сетевая интеграция
- Работа через Ethernet и оптоволокно  
- Мониторинг сетевых параметров  
- Удалённое управление  

### 🔋 Контроль питания (ИБП)
- Контроль нагрузки  
- Мониторинг температуры  
- Отслеживание заряда батареи  
- Аварийные уведомления  

### 📊 Интерфейс
- Графический интерфейс LabVIEW  
- Веб-интерфейс мониторинга  
- Отображение параметров в реальном времени
