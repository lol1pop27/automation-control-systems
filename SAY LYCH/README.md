🚀 SAY LYCH

Система синхронизации, управления и мониторинга измерительного комплекса

📌 Описание

SAY LYCH — программно-аппаратная система, разработанная в LabVIEW, предназначенная для централизованного управления, синхронизации и мониторинга распределённого измерительного комплекса.

Система объединяет:

⏱ высокоточную временную синхронизацию сигналов

📡 сбор и обработку аналоговых данных

🔌 управление дискретными каналами

Взаимодействие между компонентами реализовано через промышленную сеть с использованием оптоволоконной линии связи.

🧠 Технологический стек

🧩 LabVIEW (National Instruments) — основная среда разработки

🌐 Ethernet / TCP-IP

🔗 Оптоволоконная сеть
