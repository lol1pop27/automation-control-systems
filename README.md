🚀 Measurement Control Systems Suite
Комплекс систем синхронизации, управления и мониторинга измерительных установок
📌 Описание

Данный репозиторий объединяет несколько программно-аппаратных систем, разработанных для управления, синхронизации и мониторинга измерительных комплексов.

Системы применяются в научных и инженерных установках и обеспечивают:

⏱ точную временную синхронизацию
📡 сбор и обработку данных
🔌 управление исполнительными устройствами
🌐 удалённый контроль через сеть
🧩 Состав комплекса

🔹 SAY LYCH

Система синхронизации и управления измерительным комплексом, разработанная в LabVIEW.

Возможности:

управление генератором задержек (DG645)
сбор аналоговых данных (ADAM-4017)
управление дискретными каналами (ADAM-4068)
работа через Ethernet и оптоволокно

🔹 NetAgent

Веб-ориентированная система мониторинга и управления оборудованием.

Возможности:

удалённый контроль параметров системы
визуализация данных
управление каналами и устройствами
мониторинг питания и сети

🔹 САУ РЗДРТ

Система управления шаговыми двигателями рентгеновского стенда «РЭКС».

Возможности:

позиционирование оптических элементов
управление через RS-232 (COM-порт)
работа с ПО Shag4.exe
высокая точность позиционирования
