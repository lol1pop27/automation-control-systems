## 📌 Описание

Данный репозиторий объединяет несколько **программно-аппаратных систем**, разработанных для управления, синхронизации и мониторинга измерительных комплексов.

Системы применяются в научных и инженерных установках и обеспечивают:

| Функция | Описание |
|---------|----------|
| ⏱ Точная временная синхронизация | Синхронизация сигналов с наносекундной точностью |
| 📡 Сбор и обработка данных | Измерение аналоговых сигналов в реальном времени |
| 🔌 Управление исполнительными устройствами | Дискретные каналы, шаговые двигатели |
| 🌐 Удалённый контроль через сеть | Мониторинг и управление по Ethernet/оптоволокну |

---

## 🧩 Состав комплекса

### 🔹 SAY LYCH

**Система синхронизации и управления измерительным комплексом** (LabVIEW)

| Компонент | Устройство | Функция |
|-----------|------------|---------|
| Генератор задержек | Stanford Research DG645 | Точная синхронизация сигналов |
| Аналоговый ввод | Advantech ADAM-4017 | Сбор аналоговых данных |
| Дискретный вывод | Advantech ADAM-4068 | Управление исполнителями |
| Сеть | Ethernet + оптоволокно | Удалённое взаимодействие |

**Основные возможности:**
- Управление каналами задержек A–H (наносекундный диапазон)
- Формирование и синхронизация импульсов
- Отображение данных в реальном времени
- Контроль питания и сетевых параметров

---

### 🔹 NetAgent

**Веб-ориентированная система мониторинга и управления оборудованием**

| Функция | Описание |
|---------|----------|
| 🌐 Удалённый контроль | Доступ через веб-интерфейс из любой точки сети |
| 📊 Визуализация данных | Графики, показатели датчиков, состояние устройств |
| 🎛 Управление каналами | Включение/выключение исполнительных устройств |
| 🔋 Мониторинг питания | Контроль ИБП, нагрузки, температуры |

**Основные возможности:**
- Доступ через современный браузер
- Отображение параметров в реальном времени
- Логирование событий и аварийных ситуаций

---

### 🔹 САУ РЗДРТ

**Система управления шаговыми двигателями рентгеновского стенда «РЭКС»**

| Компонент | Устройство | Функция |
|-----------|------------|---------|
| Интерфейс | RS-232 (COM-порт) | Связь с управляющим ПК |
| ПО управления | Shag4.exe | Отправка команд позиционирования |
| Исполнители | Шаговые двигатели | Позиционирование оптических элементов |

**Основные возможности:**
- Высокоточное позиционирование
- Управление через COM-порт
- Интеграция с рентгеновским стендом «РЭКС»

## 🏗 Архитектура комплекса

**Компоненты:**

| Уровень | Элементы |
|---------|----------|
| 💻 Управляющие ПК | LabVIEW-приложения, веб-сервер |
| 🌐 Сетевая инфраструктура | Ethernet, оптоволокно, коммутаторы |
| 🔗 Промышленное оборудование | DG645, ADAM-4017, ADAM-4068 |
| ⚙️ Исполнительные механизмы | Шаговые двигатели, датчики, затворы |

**Связи:**
- SAY LYCH — управление через Ethernet/оптоволокно (Modbus/TCP, SCPI)
- NetAgent — веб-доступ через браузер
- САУ РЗДРТ — управление через RS-232 (COM-порт)

## 📄 Заключение

Комплекс систем обеспечивает полный цикл управления измерительными установками:

- ✅ **SAY LYCH** — высокоточная синхронизация и управление оборудованием
- ✅ **NetAgent** — удалённый мониторинг через веб-интерфейс

Все системы интегрированы в единую инфраструктуру и готовы к применению в научных и промышленных задачах.
