## 🔹 SAY LYCH

**Система синхронизации и управления измерительным комплексом** (LabVIEW)

| Параметр | Описание |
|----------|----------|
| Генератор задержек | Stanford Research DG645 |
| Аналоговый ввод | Advantech ADAM-4017 |
| Дискретный вывод | Advantech ADAM-4068 |
| Сеть | Ethernet + оптоволокно |
| Протоколы | Modbus/TCP, SCPI |

**Возможности:**
- Управление каналами задержек A–H (наносекунды)
- Сбор аналоговых данных в реальном времени
- Управление исполнительными устройствами
- Контроль питания и сети

---

## 🔹 NetAgent

**Веб-ориентированная система мониторинга и управления**

| Параметр | Описание |
|----------|----------|
| Доступ | Веб-интерфейс (любой браузер) |
| Мониторинг | ИБП, нагрузка, температура |
| Управление | Включение/выключение каналов |

**Возможности:**
- Удалённый контроль из любой точки сети
- Визуализация данных (графики, датчики)
- Логирование событий и аварий

---

## 🔹 САУ РЗДРТ

**Управление шаговыми двигателями рентгеновского стенда «РЭКС»**

| Параметр | Описание |
|----------|----------|
| Интерфейс | RS-232 (COM-порт) |
| ПО | Shag4.exe |
| Исполнители | Шаговые двигатели |

**Возможности:**
- Высокоточное позиционирование оптики
- Управление через COM-порт
- Интеграция со стендом «РЭКС»

## 📊 Сравнение систем

| Система | Назначение | Интерфейс | Ключевое |
|---------|------------|-----------|----------|
| SAY LYCH | Синхронизация + управление | Ethernet / оптоволокно | Наносекундная точность |
| NetAgent | Мониторинг + управление | Веб-браузер | Удалённый доступ |
| САУ РЗДРТ | Позиционирование | RS-232 (COM) | Точность позиционирования |
