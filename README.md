## 🔹 SAY LYCH

**Система синхронизации и управления измерительным комплексом** (LabVIEW)

| Параметр | Описание |
|----------|----------|
| Генератор задержек | Stanford Research DG645 |
| Аналоговый ввод | Advantech ADAM-4017 |
| Дискретный вывод | Advantech ADAM-4068 |
| Сеть | Ethernet + оптоволокно |
| Протоколы | Modbus/TCP, SCPI |
|----------|----------|
| Доступ | Веб-интерфейс (любой браузер) |
| Мониторинг | ИБП, нагрузка, температура |
| Управление | Включение/выключение каналов |

**Возможности:**
- Управление каналами задержек A–H (наносекунды)
- Сбор аналоговых данных в реальном времени
- Управление исполнительными устройствами
- Контроль питания и сети
- Удалённый контроль из любой точки сети
- Визуализация данных (графики, датчики)
- Логирование событий и аварий

## 🔹 АСУ ТП вакуумирования

**Автоматизированная система управления откачкой вакуумной камеры**

**Цель:** автоматизация процесса вакуумной откачки экспериментальной установки.

**Задачи:**
1. Провести анализ вакуумной системы
2. Разработать АСУ ТП вакуумирования
3. Сформировать и обосновать элементную базу
4. Провести макетирование АСУ ТП

### Используемое оборудование

| Устройство | Модель | Назначение |
|------------|--------|-------------|
| Турбомолекулярный насос | HiPace 80 | Высоковакуумная откачка |
| Форвакуумный насос | ISP-500B | Предварительная откачка |
| Вакуумметр | WRG-S-NW25 | Измерение давления |
| Пневмозатвор | HVA 83110-0253V | Перекрытие линии |
| Шаговый двигатель | FL57STH56-1006 | Управление затвором |
| Контроллер ШД | SMSD 4.2 | Управление двигателем |
| Редуктор | AG75 | Передаточное отношение 75,5 |

### Алгоритм работы

**Запуск откачки:**
1. Проверка ИБП
2. Запуск форвакуумного насоса
3. Контроль предварительного вакуума
4. Запуск турбомолекулярного насоса
5. Открытие затвора
6. Откачка до 10⁻⁵ Торр

**Остановка откачки:**
1. Закрытие затвора
2. Останов турбомолекулярного насоса
3. Ожидание 23 мин (остывание)
4. Останов форвакуумного насоса


**Возможности:**
- Автоматический и дистанционный режимы
- Мониторинг давления в реальном времени
- Контроль состояния ИБП
- Логирование данных в файл
