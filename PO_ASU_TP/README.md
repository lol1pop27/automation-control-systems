## 📌 Описание

**АСУ ТП вакуумирования** — система управления откачкой вакуумной камеры экспериментальной установки. Поддерживает автоматический и дистанционный режимы, контроль давления и защиту от аварий.

---

## 🎯 Актуальность

- Повышение эффективности труда
- Оптимизация процесса вакуумирования
- Исключение ошибок оператора
- Защита оборудования от внештатных ситуаций

---

## 🧩 Цель и задачи

**Цель:** автоматизация процесса вакуумной откачки экспериментальной установки.

**Задачи:**

1. Провести анализ вакуумной системы экспериментальной установки
2. Разработать АСУ ТП вакуумирования
3. Сформировать и обосновать элементную базу
4. Провести макетирование АСУ ТП для обоснования решений

## 📐 Расчёт управления затвором

Количество шагов для открытия/закрытия затвора:
N = (n × 360°) / (s × k)
Параметры:
- n = 12,7 — количество оборотов для открытия/закрытия
- s = 1,8° — полный шаг двигателя
- k = 75,5 — передаточное отношение редуктора

## 🔄 Алгоритм работы

### Запуск откачки

1. Старт системы
2. Проверка состояния ИБП (при ошибке — аварийная остановка)
3. Запуск форвакуумного насоса
4. Контроль предварительного вакуума
5. Запуск турбомолекулярного насоса
6. Открытие затвора (шаговый двигатель)
7. Откачка камеры до 10⁻⁵ Торр

### Остановка откачки

1. Получение команды STOP
2. Закрытие затвора
3. Останов турбомолекулярного насоса
4. Ожидание 23 минуты (остывание турбины)
5. Останов форвакуумного насоса

> ⚠️ **Важно:** 23 минуты — минимальное время для полной остановки технологического процесса откачки (остывание турбомолекулярного насоса).

## 🏗️ Структурная схема АСУ ТП

**Компоненты системы:**
- ПК с программой LabVIEW
- Сетевой Ethernet-коммутатор
- ИБП с сетевой картой
- Вакуумметр EDWARDS WRG-S-NW25
- Контроллер шагового двигателя SMSD 4.2
- Шаговый двигатель FL57STH56-1006
- Затвор HVA 83110-0253V

**Связи:**
- Все устройства подключены через Ethernet
- ПК управляет всеми компонентами по сети
- Контроллер управляет шаговым двигателем
- Шаговый двигатель открывает/закрывает затвор

---

## 💻 Программное обеспечение

**Среда разработки:** LabVIEW

**Функционал программы:**

| Функция | Описание |
|---------|----------|
| Инициализация | Выделение сетевого адреса, настройка порта ИБП, модулей ввода-вывода |
| Мониторинг ИБП | Контроль состояния источника бесперебойного питания |
| Мониторинг вакуумметра | Отображение текущего давления в камере |
| Дистанционное управление | Ручной запуск и остановка откачки |
| Автоматическое управление | Откачка по заданному алгоритму |
| Логирование | Автоматическое сохранение данных давления в лог-файл |

**Окна программы:**
1. Мониторинг состояния ИБП
2. Мониторинг показаний вакуумметра
3. Управление откачкой (дистанционный режим)
4. Автоматическое управление
5. Настройки и инициализация

## 🧪 Испытания и результаты

**Параметры испытаний:**
- Конечное давление: 10⁻⁵ Торр
- Время остановки: 23 минуты
- Количество режимов работы: 2 (автоматический и дистанционный)
- Логирование: включено

**Результат:** Система показала работоспособность в штатном и аварийном режимах.

## 🚀 Запуск системы

1. Проверить подключение оборудования к сети Ethernet
2. Включить ИБП и убедиться в его исправности
3. Запустить программу LabVIEW
4. Выполнить инициализацию (настройка портов и адресов)
5. Выбрать режим работы:
   - Дистанционное управление — ручной запуск и останов
   - Автоматическое управление — автоматический цикл откачки
