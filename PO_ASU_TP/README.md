## 📌 Описание проекта

**АСУ ТП вакуумирования** — автоматизированная система, разработанная для управления процессом откачки вакуумной камеры экспериментальной установки. Система обеспечивает автоматический и дистанционный режимы управления, контроль давления, мониторинг состояния оборудования и защиту от аварийных ситуаций.

---

## 🎯 Актуальность

- Повышение эффективности и производительности труда
- Оптимизация технологического процесса вакуумирования
- Минимизация ошибок, связанных с человеческим фактором
- Защита оборудования от внештатных ситуаций

### Принципиальная схема вакуумной системы

```text
┌─────────────────────────────────────────────────────────┐
│                   Вакуумная камера                       │
│                    P ≈ 10⁻⁵ Торр                         │
└─────────────────┬───────────────────┬───────────────────┘
                  │                   │
      ┌───────────┴───────────┐       │
      │   Основная линия       │       │
      │   откачки              │       │
      └───────────┬───────────┘       │
                  │                   │
      ┌───────────┴───────────┐       │
      │ Турбомолекулярный      │       │
      │ насос (HiPace 80)      │       │
      └───────────┬───────────┘       │
                  │                   │
      ┌───────────┴───────────┐       │
      │   Форвакуумный насос   │◄──────┘
      │   (ANEST IWATA ISP-500B)│
      └───────────────────────┘
```
Используемое оборудование
Устройство	Модель	Назначение
🌀 Турбомолекулярный насос	PFEIFER VACUUM HiPace 80	Высоковакуумная откачка
🔧 Форвакуумный насос	ANEST IWATA ISP-500B	Предварительная откачка
📡 Широкодиапазонный вакуумметр	EDWARDS WRG-S-NW25	Измерение давления
🚪 Пневмозатвор	HVA 83110-0253V	Перекрытие вакуумной линии
⚙️ Шаговый двигатель	FL57STH56-1006	Управление затвором
🧠 Контроллер ШД	SMSD 4.2	Управление шаговым двигателем
🔌 Редуктор	AG75	Передаточное отношение 75,5
📐 Расчёт управления затвором
Количество шагов для открытия/закрытия затвора:

text
N = (n × 360°) / (s × k)
где:

n = 12,7 — количество оборотов для открытия/закрытия

s = 1,8° — полный шаг двигателя

k = 75,5 — передаточное отношение редуктора
