## 📌 Описание

**АСУ ТП вакуумирования** — система управления откачкой вакуумной камеры экспериментальной установки. Поддерживает автоматический и дистанционный режимы, контроль давления и защиту от аварий.

---

## 🎯 Актуальность

- Повышение эффективности
- Оптимизация процесса вакуумирования
- Исключение ошибок оператора
- Защита оборудования

## 📐 Расчёт шагов затвора
N = (n × 360°) / (s × k)

где:
n = 12.7 — оборотов для открытия/закрытия
s = 1.8° — шаг двигателя
k = 75.5 — передаточное отношение редуктора
